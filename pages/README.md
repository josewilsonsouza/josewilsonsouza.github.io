# Páginas do Site

Esta pasta contém todas as páginas estáticas do site.

## Arquivos

- **posts.html** - Página que lista todos os artigos/posts (`/posts`)
- **about.markdown** - Página "Sobre" com informações profissionais (`/about`)
- **contato.markdown** - Página de contato e serviços (`/contato`)
- **recursos.markdown** - Recursos matemáticos e educacionais (`/recursos`)

## Permalinks

Cada arquivo define seu próprio permalink no front matter, então as URLs permanecem limpas e organizadas mesmo com os arquivos nesta pasta.
